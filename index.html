This is my personal little guide on how to make a website

Let's start

1. Choose what to do

Identify the primary purpose of the website and its goals. This step includes determining the target audience and what actions you want visitors to take. I always prefer to make websites for practice since I'm studying right now and no one would contact me to make a website. One of the most important things is to find a website that you like and try to take inspiration for your website.

Here's some sites where you can find inspiration:

AWWARDS
Minimal Gallery
Dark Mode Design
Refero
Stack Sorted

2. Build the website in figma

Before moving to the development phase, use Figma to design the website. Figma allows for collaborative design, prototyping, and feedback collection all in one place. Create wireframes, mockups, and prototypes to visualize the design and layout of your website.

Figma website

3. Build the html and css code

Convert the Figma design into a functional website using coding languages such as HTML, CSS, and JavaScript. This step may involve using a content management system (CMS) like WordPress, or building the site from scratch.

You can use some IDE for this step, below are some of my favorites

vs code
web storm
php storm
Eclipse

4. Publish

After testing and final revisions, make the website live by publishing it on the web. Ensure all domain and hosting settings are correctly configured.

To publish a website you need to pay a Web Hosting Service such as Aruba, so you can register a domain name like pietrotombaccini.com

Now enjoy your new website and remember to keep it updated!
